## Talk GraphX
This is the repository for the talk about GraphX held for Codemotion Milan conference. 
To compile and run, you have to add the spark-assembly-1.5.1-hadoop2.6.0.jar file located
in "lib" directory of Apache Spark v1.5.1 tarball (the one "Pre-built for Hadoop 2.6 and later").

Scala version is 2.10.4.
